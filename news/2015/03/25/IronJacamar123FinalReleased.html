IronJacamar 1.2.3.Final released

March 25th, 2015

I'm proud to announce the IronJacamar 1.2.3.Final release. Full release notes are here .

Changes

This release is mainly a bug fix release, but contains some major changes to the tracer as well,

  • [JBJCA-1252] MCPs can become detached
  • [JBJCA-1250] TotalPoolTime / TotalUsageTime not always updated
  • [JBJCA-1251] Only allow certain decrementer policies for CRI pools
  • [JBJCA-1254] Tracer improvements

Due to JBJCA-1252, all users should upgrade their installations.

The Road Ahead

This release took some time away from IronJacamar 2 development, but the repository keeps seeing updates, so it is progressing.

Enjoy !

On behalf of the IronJacamar community,
    Jesper
